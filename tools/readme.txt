scripts used for FTC SW development:
-----------------
w.cmd: run adb over wifi
You need to connect your phone's your WIFI router which your PC connects to.
This script figures out the IP address of your android phone and start adb on TCP link;

------------------
logc.cmd: logcat for robot controller process
It figures out robot controller's process ID and only log prints from that process
you can redirect it to a file or log on screen with color
